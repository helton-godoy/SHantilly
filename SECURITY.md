# Política de Segurança

## Versões Suportadas

Atualmente, oferecemos suporte de segurança para as seguintes versões:

| Versão | Suportada          |
| ------ | ------------------ |
| 1.0.x  | :white_check_mark: |
| < 1.0  | :x:                |

## Reportando uma Vulnerabilidade

Levamos a segurança a sério. Se você encontrou uma vulnerabilidade, por favor, **NÃO** abra uma Issue pública.

1.  Envie um e-mail para **security@shantilly.org** (ou entre em contato direto com o mantenedor @helton-godoy).
2.  Inclua detalhes sobre a vulnerabilidade e passos para reprodução.
3.  Responderemos o mais rápido possível (geralmente em até 48h) para confirmar o recebimento.
4.  Trabalharemos em uma correção e lançaremos um patch de segurança.

Agradecemos por ajudar a manter a comunidade segura!
