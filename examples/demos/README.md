**Diretório `demos`:**

Este diretório é dedicado a scripts de demonstração, com o objetivo de fornecer exemplos práticos em formato de prova de conceito para aplicações mais robustas, integrando diversos elementos do showbox de forma harmoniosa, demonstrando sua versatilidade e potencial em cenários reais.

As demonstrações devem ser concisas e focadas em ilustrar possibilidades reais de implementação para cenários práticos, destacando a abordagem padrão e mais didática para combinar os elementos, facilitando o aprendizado e a replicação por desenvolvedores de scripts que utilizarão o showbox.
